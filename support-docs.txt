Support Docs – Optizen AI Pro & Enterprise

/how-to-install-optizen/ 

Currently, to use advances features of Optizen PRO and Enterprise, we will need to manually install these features. Please send a support request via our Contact form, with your .myshopify.com URL. 

We then make a limited permission access request and complete the installation.

If you plan on only using the AI feature, the standard install from the Shopify app store will work out of the box.

/using-the-sync-button/ - There may be times where you can’t find the tag or collection you want, when using the search function in Optizen. To keep our data bases clean, new tags and collections need to be synced to the app.

There are two “Sync” button. One under the Collections tab, and one under the Tags tab. Each button serves the same purpose. Once clicked, it will sync any new collections and tags to the app.

Click the “Sync” button to sync new tags and collection. Depending on the size of your store, it may take a few minutes.
